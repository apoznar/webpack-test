Strona główna HTML CSS JS Inne notatki

Moje notatki z uczestnictwa w 14-tygodniowym kursie "Co Ten Frontend" prowadzonym przez Macieja Korsana.

Od początku dnia minęło:

HTML

HTML - lekcja I

Dokument HTML dzielimy na części head i body. W części body możemy wyróżnić części main, section, article, header, footer. Możemy stworzyć także nagłówki h1-h6 oraz paragrafy - p.

HTML - lekcja II

Samozamykający znacznik br/ rozpoczyna nową linię.
Znacznik b pogrubia, a i tworzy kursywę. Znaczniki strong i em działają odpowiednio podobnie, ale mają za to znaczenie semantyczne. Inny znacznik 'a' może zawierać linki/łącza wewnątrz dokumentu, ma atrybuty href="" i target="". W href podajemy adres, do którego prowadzi klikniecie w znacznik. Atrybut w elemencie id="moj-identyfikator" identyfikuje element, wprowadzenie go do znacznika a jeżeli zapiszemy go w href="#moj-identyfikator" po kliknięciu przenosi do oznaczonego miejsca w obrębie strony . Znacznik nav tworzy nawigację strony - w części body tworzymy header z nav (zaraz nad mainem aby przenosić się do różnych stron powiązanych linkiem. Inny znacznik - img z atrybutami src i alt pozwala nam na dołączenie dowolnego obrazka z podanego źródła i tekstem alternatywnym.

Życie w pudełku

Do opakowania treści używamy znacznika div z klasą box. Elementy pudełka od zewnątrz: margin (margines), border (obramowanie), padding (wewnętrzny margines) oraz element.

HTML - lekcja III

Lista nieuporządkowana - ul, listowanie przez li. Uprządkowana - ol. list-style(-type) to styl listy. Dl - description list. Dt - description term. DD - description details. Table - tabela, w niej wiersze tr i kolumny td. Nagłówek tabeli thead, komórka w nagłówku th. Ciało tabeli - tbody. Caption - opis tabeli.

Formularze

Znacznik form i elementy input (w znaczniku p) - ten może mieć atrybut name, id. Znacznik label tworzy opis pola, atrybut for u niego powinien odnosić się do id form aby był połączony. Label może też okalać form - jako drugi sposób. Element input może mieć type=text, value (domyślna wartość), placeholder (tekst podpowiedzi, nie zastępuje atrybutu label). Możemy nadać im klasy i zrobić elementom display block i width 100%. Znacznik fieldset to grupa pól a legend to opis tej grupy. Typ radio z inputa występuje z name, trzeba opakować w labelu z nazwą. Typ checkbox, może mieć wartość checked oraz type(number, tel, email, password, file), range (obok min, max i step; możemy przez DOM sprawdzić wartość). Element select tworzy listę do wyboru z elementami option (mogą być z value). Znacznik textarea służy do publikowania większych treści. Input type submit wysyła formularz. Aby wysyłać formularze dodajemy do form atrybut action z url gdzie go wysyłamy. Żądanie typu GET dopisuje zebrane dane do adresu (bierze atrybut value)- jest domyślne. Metoda POST wysyła wartości w formie form data bez uwidocznienia w adresie. Mamy jeszcze wartość enctype i ona mówi czy to jest zwykły formularz czy coś jeszcze jest na tej stronie (np. pliki). Poprzez stronę dev.cotenfrontend.pl można sprawdzić uwidocznienie danych obiema metodami. Atrybut required wymaga podania danych. Atrybut pattern pozwala na używanie wyrażeń regularnych.
